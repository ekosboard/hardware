# EKOS Hardware

> Open-source hardware design for the EKOS connected e-paper display.

![EKOS Device](https://ekosboard.com/assets/desktop_DSCF7619-CdRnGBlO.avif)

---

## About This Repository

This repository contains the official hardware design files for EKOS.

Currently included:
- Schematic (KiCad project)
- PCB layout (KiCad project)

Additional hardware files, such as 3D CAD models, STEP files, or FreeCAD projects, will be added in the future.
